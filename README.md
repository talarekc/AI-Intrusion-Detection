# AI-Powered Network Intrusion Detection & Alert Dashboard

## Project Overview
This project is a cybersecurity capstone focused on detecting suspicious network activity using machine learning. The system will analyze network traffic data, classify it as benign or malicious, and display alerts in a dashboard.

## Team Members and Roles
- **Cole** — ML model, preprocessing, and detection pipeline
- **Brian** — Streamlit dashboard and integration
- **Frank** — Research, documentation, attack scenarios, and presentation

## Planned Features
- Train and evaluate a machine learning model on network traffic data
- Detect common attacks such as port scans, brute force attempts, DDoS, and web attacks
- Build a pipeline that processes new data and produces predictions
- Display alerts in a dashboard with filters and visualizations
- Evaluate performance using precision, recall, F1 score, and false positive rate

## Project Structure
- `data/` — local dataset storage only, not uploaded to GitHub
- `docs/` — project notes, diagrams, and documentation
- `models/` — saved trained models
- `notebooks/` — exploration and experimentation
- `src/` — main source code

## Dataset
This project plans to use the CICIDS2017 dataset.

Because the dataset is large, it is not stored in this repository. Each team member should download it separately and place it in the `data/` folder on their own machine.

## Setup Notes
1. Clone the repository
2. Download the dataset separately
3. Place the dataset inside the `data/` folder
4. Keep large files and raw data out of GitHub
